Get Countrysweb running on your local machine in just a few minutes.

## Prerequisites

Before installing Countrysweb, ensure you have the following installed:

- **Node.js** (version 18 or higher recommended)
- **pnpm** (version 10.15.0 or higher)

<Note>
This project uses pnpm as its package manager. While you can use npm or yarn, pnpm is recommended for consistency with the project configuration.
</Note>

## Installation Steps

<Steps>
  <Step title="Clone the repository">
    Clone the Countrysweb repository to your local machine:
    
    ```bash
    git clone https://github.com/yourusername/countrysweb.git
    cd countrysweb
    ```
  </Step>

  <Step title="Install dependencies">
    Install the required dependencies using your preferred package manager:
    
    <CodeGroup>
    ```bash pnpm
    pnpm install
    ```

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```
    </CodeGroup>

    The project includes the following key dependencies:
    
    - **React** (v19.2.0) - UI framework
    - **React Router DOM** (v7.13.0) - Client-side routing
    - **Axios** (v1.13.5) - HTTP client for API requests
    - **Vite** (v7.3.1) - Build tool and dev server
  </Step>

  <Step title="Start the development server">
    Run the development server:
    
    <CodeGroup>
    ```bash pnpm
    pnpm dev
    ```

    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```
    </CodeGroup>

    The application will be available at `http://localhost:5173` by default.
    
    <Note>
    The dev server runs with the `--host` flag, making it accessible from other devices on your network.
    </Note>
  </Step>

  <Step title="Verify installation">
    Open your browser and navigate to `http://localhost:5173`. You should see the Countrysweb homepage with the search interface.
  </Step>
</Steps>

## Additional Scripts

The project includes several useful npm scripts:

- `pnpm build` - Build the application for production
- `pnpm preview` - Preview the production build locally
- `pnpm lint` - Run ESLint to check code quality
