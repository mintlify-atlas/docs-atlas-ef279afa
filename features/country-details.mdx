# Country Details

The country details page displays comprehensive information about each country, including flags, coat of arms, geographical data, and demographic information.

## Widget Component Pattern

The country details use a reusable `Widget` component to display information consistently:

```jsx
const Widget = ({ icon, caption, value }) => {
  return (
    <div className="column">
      <h3>{caption}</h3>
      <p className="row">
        {icon}
        {value}
      </p>
    </div>
  );
};
```

## Population Formatting

The `formatPopulation` utility function formats large numbers with comma separators for better readability:

```javascript
const formatPopulation = (population) => {
  return population.toString().replace(/\B(?=(\d{3})+(?!\d))/g, ",");
};
```

<Note>
  This regex-based formatter adds commas every three digits from right to left. For example: `123456789` becomes `123,456,789`
</Note>

### Formatting Examples

| Raw Value | Formatted Value |
|-----------|----------------|
| 1000 | 1,000 |
| 50000000 | 50,000,000 |
| 331002651 | 331,002,651 |

## Dynamic Region Icons

The application displays region-specific globe icons based on the country's continent:

```jsx
const getContinentIcon = () => {
  const continents = country.region;
  if (continents.includes("Europe") || continents.includes("Africa")) {
    return <GlobeEurope />;
  }
  if (continents.includes("Asia") || continents.includes("Australia")) {
    return <GlobeAsia />;
  }
  return <GlobeAmericas />;
};
```

## Languages Display

Countries with multiple official languages display all languages with their ISO codes:

```jsx
<h3>Idiomas:</h3>
<ul className="languages-list">
  {Object.entries(country.languages).map(([key, lang], index) => {
    return (
      <li key={index}>
        <p>{`${lang} (${key})`}</p>
      </li>
    );
  })}
</ul>
```

<Note>
  The languages object contains key-value pairs where the key is the ISO language code and the value is the language name.
</Note>

### Language Display Example

For Switzerland, the languages would display as:
- French (fra)
- German (deu)
- Italian (ita)
- Romansh (roh)

## Border Countries Section

The details page includes a dedicated section for neighboring countries:

```jsx
<h3>Fronteras:</h3>
{!country.borders && <p>No tiene fronteras</p>}
{country.borders && (
  <ul className="borders-list">
    {country.borders.map((border, index) => {
      const bddata = countries.find((c) => c.cca3 === border);
      return <BorderCountry key={index} country={bddata} />;
    })}
  </ul>
)}
```
